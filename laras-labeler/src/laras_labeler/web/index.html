Feature set —
Which set for which behavior? — a starting point; the only real test is to try a couple and compare AP/F1 (📊 Stats). Rules of thumb:
If the behavior is defined by…exampletry
being at a landmark (needs a spout/ROI set)drinkingSpout only
position at the cage wall/rim (needs a cage ROI)climbing, rearing on rimNo social (cage + pose beats cage-only)
body motion/posture alone, no landmarkgrooming, freezing, locomotionPose only
interaction between two micefighting, following, sniffingSocial + pose
not sure yet—No social, then trim down
Leaner usually beats “All”: extra columns can overfit & dilute a clean signal. Adding a feature that looks discriminative alone (📈 Feature diff) may still not help if it’s redundant with signals already in the set — always confirm with AP/F1.
How the stats are computed — cross-validated with StratifiedGroupKFold where each CV group is a labeled bout, so adjacent frames never leak between train and test (an honest, not-optimistic estimate). AP = area under the precision–recall curve (threshold-free). Frame P/R/F1 are at probability ≥ 0.5: precision = of frames the model calls the behavior, how many you labeled it; recall = of your positive frames, how many it caught. Bout P/R scores whole bouts — a labeled bout counts as caught if its mean cross-validated probability ≥ 0.5. Needs ≥ 2 labeled bouts per class to report.

Tips & keyboard shortcuts

Workflow

Import prior annotations (or paint a few) → Train → the model proposes bouts → Review each one (the timeline zooms to it): [/] trim the accept range at the playhead, y accept · n reject · ⏎ next, or paint the zoomed lane to fix specific frames → Train again to refine.

Tracks & behaviors

Labels & predictions are kept separate per track. Pick a track by clicking a chip, keys 1–9, Tab to cycle, or clicking the animal in the video. Everything you paint, train, and review applies to the selected track.

Painting labels

Press a behavior's hotkey to start → move the playhead → press again to commit; or drag a lane. blue = Happening (p), red = Not-happening (n), ▨ gray = Unknown (u) (saved, not trained). Remove with Select → Delete.

Tools

✏ Paint (b) drags a lane to label. ☝ Select (v): drag along the bar to scrub to a bout, click a bout to select + play it (then drag the ▮ handles or [ / ] to adjust, Delete/🗑 to remove). Picking a paint action switches back to Paint.

Video & keys

Double-click the video to zoom on a spot (again to reset), then drag to recenter. Also ＋ / − / 0 or Ctrl/⌘+scroll. Ruler = scrub · ←/→ step (Ctrl ±30) · Space play · Esc cancel · Ctrl+Z undo.
